# Xuper-CA

[![License](https://img.shields.io/github/license/xuperchain/xuperchain?style=flat-square)](/LICENSE)

支持xchain通过CA证书控制全节点的权限

### 环境配置

* 操作系统：支持Linux以及Mac OS
* 开发语言：Go 1.13.x及以上
* 版本控制工具：Git

### 构建

克隆Xuper-CA仓库
```
git clone https://github.com/xuperchain/xuper-ca
```

编译CA Server
```
make
```
